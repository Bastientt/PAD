# PAD
utilisation de mediapipe pour l'ia:
    - plus précise pour la reconnaissance faciale
    - plus simple pour le reconnaissance du BG
    
How does it works:
    - lancer ia_pad.py
    - l'ia check le visage les mouvements et le background
    - renvoie le nombres d'erreurs durant la video
    - renvoie le pourcentage de veracité de la video
